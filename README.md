# Miriam Aguiar Odontologia

Landing page desenvolvida em React + TypeScript para a clínica odontológica, com identidade visual inspirada nas imagens da marca.

## Tecnologias usadas

- React 19
- TypeScript
- Vite
- Tailwind CSS 4

## Como rodar o projeto

### 1. Instalar as dependências

```bash
npm install
```

### 2. Iniciar o servidor de desenvolvimento

```bash
npm run dev
```
